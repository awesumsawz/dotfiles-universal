Register a <link rel="preload" as="image"> for the hero WebP in wp_head at priority 1

6. Add strategy => 'defer' to all registered page scripts

Update wp_register_script() and wp_enqueue_script() calls for page bundles, Slick, and the combined script to use ['in_footer' => true, 'strategy' => 'defer']

Prevents layout shift and lets the browser pre-allocate space before images load

Move the hardcoded <script src="//widget.trustpilot.com/..."> out of the template and load it only on scroll/interaction via JS
