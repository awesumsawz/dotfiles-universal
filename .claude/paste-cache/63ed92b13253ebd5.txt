## Context and Goal

Duplicate the Enhanced PDP (`mens-opt/testosterone-1`) to a new Forbes-targeted variant at `mens-opt/forbes/testosterone`. Add the Forbes linkbar banner and update one line of copy. Intended for A/B testing Forbes ad traffic against the Enhanced PDP.

**PHP template changes (vs source `page-mens-opt-enhanced-pdp.php`):**

1. Template header — change template name and slug:
   - `// Template Name: Mens OPT Forbes Testosterone`
   - `$template_name = 'mens-opt-forbes-testosterone';`

2. Add Forbes linkbar — add `$linkbar_vars` to global declaration, set value, and include template part after `get_header()`:
   - `$linkbar_vars = ['linkbar_content' => 'Seen in Forbes Health Best TRT Clinic of 2025'];`
   - `require_once get_template_directory() . '/template-parts/linkbar.php';`

3. Copy update in the `.alternatives` section:
   - From: "Needle-free testosterone options available with the At-Home Testosterone Testing Kit"
   - To: "Needle-free testosterone options available with the Testosterone Test"

**SCSS changes:** Replace root selector `.mens-opt-enhanced-pdp` → `.mens-opt-forbes-testosterone` throughout, and add linkbar text color:
```

    color: #0e0b20;
}
```
